# 计划-定时归档任务-新建页优化

## 目标
- 提升新建定时归档任务页面的可用性与校验完整度。
- 清理调试输出，避免生产环境噪声日志。
- 降低目标表检查的状态过期风险。

## 待办清单
- [ ] 移除页面中的 Console 调试输出，接入规范日志或彻底去除。
- [ ] 统一批次大小输入方式，避免 Radio 与输入框双向绑定冲突。
- [ ] 补充任务名称“不可纯数字”校验，并对目标表/架构名称做基本合法性校验。
- [ ] 增加 Cron 表达式格式校验（使用 Cronos）。
- [ ] 目标表检查/创建增加防过期机制（请求版本号或输入快照）。
- [ ] 重置操作时清理目标表检查状态。

## 验收标准
- 保存前校验能明确提示错误原因，且不出现“保存后失败”的明显情况。
- 批次大小设置不产生 UI/数据不一致。
- 目标表检查结果与当前输入一致，不出现过期提示。
